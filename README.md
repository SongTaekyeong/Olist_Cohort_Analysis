# Olist 코호트 분석

> 📝 본 프로젝트는 현재 통합·고도화 작업 중입니다. 
> Sensitivity analysis 검증, 통계 검정, 예측 모델링을 추가한 업데이트 버전을 준비 중입니다.

## 프로젝트 개요
- 목적: 브라질 이커머스 고객의 재구매 패턴 분석
- 데이터: Kaggle Olist 브라질 이커머스 데이터셋
- 환경: BigQuery

## 분석 과정
- Step 1: 고객별 첫 구매월 추출
- Step 2: 고객별 재구매 개월 차이 계산
- Step 3: 코호트별 재구매 고객 수 집계
- Step 4: 코호트별 리텐션율 계산

## 인사이트

1. **낮은 재구매율**: 첫 구매 후 1개월 내 재구매율이 평균 5%로,
   95%의 고객이 1회성 구매에 그침

2. **급격한 이탈**: 2개월 후 리텐션율이 0.1%로 급감하며
   대부분의 고객이 첫 구매 후 이탈

3. **초기 데이터 주의**: 2016년 12월은 샘플 수가 적어
   통계적 신뢰도가 낮으므로 해석 시 주의 필요

## 액션 아이템
- 첫 구매 후 1개월 이내 재구매 쿠폰 발송으로 리텐션 개선 시도
